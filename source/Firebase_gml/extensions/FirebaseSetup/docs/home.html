Home

Firebase is an app development platform that helps you build and grow apps and games users love. This extension pack includes all the available ports of the modules available for the GameMaker engine. Backed by Google and trusted by millions of businesses around the world.

This extension's functionality is split up in multiple modules, each in its own extension. The extensions to import depend on the functionality that your project uses.

Note

In the latest update, importing modules necessitates the inclusion of the FirebaseSetup extension. This extension is a straightforward setup tool that is responsible for incorporating dependencies and aiding in project setup (refer to the guide section for guidance on configuring your project).

Modules

This is a list of the modules included in the Firebase collection:

  • Analytics
  • App Check
  • Authentication
  • Cloud Functions Overview
  • Cloud Messaging
  • Cloud Storage Overview
  • Crashlytics
  • Performance Monitoring
  • Remote Config
